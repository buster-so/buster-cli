# Buster CLI

The official command-line interface for Buster.

## Installation

### Quick Install (Recommended)

#### macOS/Linux
```bash
curl -fsSL https://platform.buster.so/install | bash
```

#### Windows (PowerShell)
```powershell
irm https://platform.buster.so/install | iex
```

### Manual Installation

Download the latest release for your platform from the [releases page](https://github.com/buster-so/buster-cli/releases).

## Usage

```bash
buster --help
```
